# task-manager-revisa

Task manager do desafio técnico para Revisa Hub

O banco de dados foi feito em SQlite 
backend foi feito em Fastify e prisma 
frontend foi feito em react vite, typescript, html e tailwind css.

para executar o codigo deve-se inserir a pasta no disco C: e abrir 3 terminais no VScode

no primeiro executar cd "C:\task-manager-revisa-main\task-manager-revisa-main"
"npm install"

No segundo terminal digite cd backend + Enter e em seguida execute "npm run dev:api".

No terceito terminal digite cd frontend + Enter e em seguida execute "npm run dev:web".
